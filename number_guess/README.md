# 숫자 야구

2~10 사이에서 원하는 갯수를 정하면 0~9 사이의 수를 랜덤하게 컴퓨터가 선택
이후 입력하는 수에 따라 자리가 맞는 수는 s, 수는 있느나 자리가 안 맞는 수는 b로 표시
각 수의 자리까지 모두 맞춰서 입력할 경우 정답 처리
이후 원한다면 기록을 저장 및 특정 닉네임이나 기록 순으로 기록 검색 가능
저장소는 언제든지 변경 가능

ex)
3 입력
컴퓨터가 정한 수 1 5 7

사용자: 2 7 4
컴퓨터: 0s 1b
사용자: 1 5 9
컴퓨터: 1s 0b
사용자: 1 5 7
컴퓨터: 정답입니다. 3번만에 맞추셨군요 (신기록인 경우 신기록입니다) 기록을 저장하시겠습니까?
사용자: yes
컴퓨터: 함께 저장될 닉네임을 입력해주세요.
사용자: kim1
컴퓨터: 저장되었습니다.


## 기술 구현부
기록 파일은 객체를 바이너리 파일로 작성
전체 기록 및 특정 닉네임에서의 기록순 정렬 보기

## 추가 개발 사항
사용자가 문제를 내고 컴퓨터 level을 정해서 대결하는 부분
